Returns a map of helm requirements for a given upstream environment value

For example, given environment variables:
  SOMETHING: 'hello',
  HELMPARAM_version: '4.2',
  HELMREQ_loadbalancer: '1.4.5'
  HELMREQ_other: '1.4.6'

And a call to unpackReqVars:

  helmReqs = unpackReqVars {
    upstreamEnv = env
  }
  println helmReqs

The result would be something like:

  {loadbalancer=1.4.5, other=1.48}
