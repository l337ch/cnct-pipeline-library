Unpacks passed in upstream environment into a helm --set string
Environment variables must start with HELMPARAM_
_'s are replaced with .'s

For example, give environment variables:
  SOMETHING: 'hello',
  HELMPARAM_version: '4.2',
  HELMPARAM_description: 'blah blah blah',
  HELMPARAM_images_agentImage: 'gcr.io/blah/agent:12345',
  HELMPARAM_images_masterImage: 'gcr.io/blah/master:12345',
  HELMPARAM_nodePorts_nginx: 32000,
  HELMPARAM_nodePorts_jenkins: 32004,
  HELMPARAM_services_backend_proxy: 'something something',
  HELMPARAM_services_other: 'darkside',
  HELMREQ_loadbalancer: '1.4.5'

And a call to unpackDependencyVars:

  helmVars = unpackDependencyVars {
    upstreamEnv = env
  }
  println helmVars

Might result in something like:

  version=4.2,description=blah blah blah,images.agentImage=gcr.io/blah/agent:12345,images.masterImage=gcr.io/blah/master:12345,nodePorts.nginx=32000,nodePorts.jenkins=32004,services.backend.proxy=something something,services.other=darkside
